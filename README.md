# Wormy (Snake игра со динамички ѕидови)
Оваа апликација претставува едноставна 2D игра инспирирана од класичната Snake (Nibbles), развиена со користење на Pygame библиотеката во Python. Играчот управува со „црв“ (worm) кој се движи низ мрежа, собира јаболка и избегнува судири.

## Карактеристики
* Класична Snake механика
* Генерирање на случајни јаболка
* Динамички ѕидови (нова функционалност)
* Пауза при удар во ѕид (без губење веднаш)
* Score систем
* Start и Game Over екрани

## Динамички ѕидови (нова додадена функционалност)
Во оваа верзија е додадена нова механика:

Се генерираат случајни ѕидови на мапата
* Ѕидовите се менуваат секој пат кога ќе се изеде јаболко (нов левел)
* Ако играчот удри во ѕид:
  * играта се паузира
  * играчот може да продолжи со промена на насока
* Ѕидовите не се појавуваат врз:
  * телото на црвот
  * јаболкото

## Логика на играта
Црвот се движи во мрежа составена од клетки
Секое јаболко ја зголемува должината
По секое јаболко:
* се генерира ново јаболко
* се генерираат нови ѕидови
Играта завршува ако:
* удриш во граница
* удриш во самиот себе

## Структура на кодот
Главни функции:
* main() – иницијализација и главен loop
* runGame() – главна логика на играта
* generateWalls() – генерирање на ѕидови
* drawWorm(), drawApple(), drawWalls() – цртање елементи
* showStartScreen() и showGameOverScreen() – UI екрани
